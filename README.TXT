Crear un programa en Java que gestione el registro de estudiantes de una universidad. El programa
debe permitir agregar nuevos estudiantes con su información básica (nombre, apellido, edad,
número de estudiante), listar todos los estudiantes registrados y buscar estudiantes por su código
de estudiante.
Requerimientos:
Agregar Estudiante: Permitir al usuario ingresar la información de un nuevo estudiante (nombre,
apellido, edad, número de estudiante) y agregarlo al registro (Valor: 1.0)
Listar Estudiantes: Mostrar en pantalla la información de todos los estudiantes registrados (Valor:
1.0)
Buscar Estudiante por Número de Estudiante: Permitir al usuario ingresar un número de estudiante
y mostrar en pantalla la información del estudiante correspondiente, si existe (Valor: 1.0)
Salir del Programa: Terminar la ejecución del programa (Valor: 0.5)
Requisitos adicionales: (Valor: 1.5)
Utilizar clases y objetos para representar el modelo del mundo del problema
Utilizar estructuras de datos adecuadas para almacenar los estudiantes (por ejemplo, ArrayList)
Manejar adecuadamente casos donde el usuario ingresa información incorrecta o el número de
estudiante buscado no existe
